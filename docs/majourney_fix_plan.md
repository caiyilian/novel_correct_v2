# 魔女之旅全卷「」配平修复方案

> 目标：将 data/魔女之旅/ 中所有不配平的卷修复为完全配平
> 原则：每阶段工作量小，可独立执行，可回滚

## 阶段划分

### Stage 1: 全卷审计

**目标**: 对 5 个不配平卷做详细审计，定位每个不配平点的具体位置和上下文

**执行**:
- 对每个文件，输出所有 `「` 和 `」` 的累积配平偏差位置
- 标记每个偏差点的段落号、上下文、可能原因
- 分类：
  - **真不配平**: `「` 或 `」` 缺失/多余
  - **跨段对话**: 对话跨段落导致段内配平偏差（正常现象）
  - **符号误用**: 使用了其他引号符号（`""` `''` `【】` 等）

**产出**: `output/majourney_audit_report.md`

**验收**: 每个不配平点都有明确的位置、上下文、分类

---

### Stage 2: 修复第一卷（1 对）

**目标**: 修复第一卷 -1 偏差

**执行**:
- 基于 Stage 1 审计结果，定位偏差位置
- 判断是 `「` 缺失还是 `」` 多余
- 修复并验证

**验收**: 第一卷 2090/2090 或 2091/2091 配平

---

### Stage 3: 修复第十六卷（1 对）

**目标**: 修复第十六卷 +1 偏差

**执行**:
- 定位偏差位置
- 判断修复方式（补 `」` 还是删 `「`）
- 修复并验证

**验收**: 第十六卷配平

---

### Stage 4: 修复第十五卷 短篇集（4 对）

**目标**: 修复第十五卷短篇集 -4 偏差

**执行**:
- 逐对定位偏差
- 分析每对是缺失还是多余
- 逐对修复并验证

**验收**: 第十五卷短篇集配平

---

### Stage 5: 修复番外 莉莉艾尔与祈祷之国（4 对）

**目标**: 修复番外 +4 偏差

**执行**:
- 逐对定位偏差
- 分析每对是缺失还是多余
- 逐对修复并验证

**目标**: 修复第五卷前 7 个偏差

- 逐对修复
- 验证

**验收**: 第五卷偏差从 +22 降低

---

### Stage 7: 修复第五卷 batch 2（中 7 对）

- 验证

**验收**: 第五卷偏差继续降低

**目标**: 确认所有卷配平

**执行**:

- 确认全部 25 个文件配平
- 输出最终报告

**验收**: 全部文件「」配平，exit code = 0

---

## 执行原则

1. **每阶段独立**: 每阶段完成后可暂停，不依赖后续阶段
2. **最小修改**: 每阶段只修改一个问题，不顺手修其他
3. **验证后再下一阶段**: 每阶段完成后跑配平检查，确认修复后再进入下一阶段
4. **不插入答案文本**: 只修复符号包裹，不新增小说正文
5. **记录修改**: 每个修改记录 offset、原文、改后文

## 工具

- `tools/final_audit.py` — 全卷配平检查
- `tools/audit_detector_gap.py` — 偏差定位
- `tools/apply_polish_decisions.py` — 修复应用
